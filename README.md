# Tourist-website

This is a modern and responsive Travel Guide Website developed using HTML and CSS. The project is designed to help users explore travel destinations, hotels, and tourism-related information through an attractive and user-friendly interface.

The website includes a clean navigation bar, search functionality, destination sections, interactive buttons, hover animations, background images, and responsive layouts for a better user experience.

Features:
✔ Modern UI Design
✔ Responsive Layout
✔ Navigation Menu
✔ Background Images
✔ Search Bar Design
✔ Tourist Destination Sections
✔ Styled Buttons with Hover Effects
✔ CSS Transitions & Animations
✔ Clean and User-Friendly Interface

Technologies Used:
- HTML5
- CSS3

Purpose:
This project was created to improve frontend web development skills and practice designing modern travel websites using HTML and CSS.
